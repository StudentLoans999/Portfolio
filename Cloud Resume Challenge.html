Cloud Resume Challenge

Cloud (CRC)



Summary

  1. Created a resume written in HTML, styled in CSS and deployed online as an AWS S3 static website
  2. The website URL uses HTTPS for security with AWS CloudFront
  3. Pointed a custom domain name to the CloudFront distribution by using AWS Route 53
  4. The resume webpage includes a visitor counter programmed through JavaScript and using AWS DynamoDB database to update the visitor count
  5. Used a Lambda Function URL to communicate directly with DynamoDB
  6. Wrote a Lambda function to further advance the functionality of the visitor counter by using Python (including tests)
  7. Configured all the resources using Terraform
  8. Created a GitHub repository for my backend code to do continuous integration and deployment (CI/CD)
  9. Set up GitHub Actions to push updates to the Terraform template/Python code, so that the Python tests got run and passed
  10. Packaged and deployed the Terraform application to AWS
  11. Created a second GitHub repository for my website code, and used GitHub Actions so that when I pushed new website code, the S3 bucket automatically got updated
